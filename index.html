Organize E-Book Data for Mail Merge

Description

This suite of Google Apps Script functions is designed to facilitate the organization of e-book data for use in mail merge operations. The scripts can extract and format data from a Google Sheets document, preparing it for personalized email campaigns via Gmail.

Available Functions

showAuthorizationDialog()

Triggers the authorization flow for the script.

onOpen()

Creates a custom menu in the Google Sheets UI when the spreadsheet is opened.

outputItemsForConfirmEmail()

Processes data for confirm email mail merge.

arrangeName_(name)

Arranges a provided name into a specific format.

outputItemsForLinkShareEmail()

Processes data for link share email mail merge.
